# Dhatri Pothula Academic Portfolio

This is a clean academic portfolio website for a first-year Data Science student and aspiring Data Analyst.

## Sections

- Home
- About
- Skills
- Projects
- Certificates
- Resume
- Contact

## How to Use

1. Replace the initials placeholder with a professional photo in the `images` folder.
2. Add certificate images to the `certificates` folder.
3. Update certificate issuers, dates, and credential links in `index.html`.
4. Add your email, Kaggle profile, and LinkedIn profile.
5. Replace planned project cards with real GitHub repository links as projects are completed.
6. Upload the folder contents to your `cherrykim7538.github.io` repository.

## Deployment

Use GitHub Pages with the repository name:

```text
cherrykim7538.github.io
```

Your portfolio URL will be:

```text
https://cherrykim7538.github.io/
```
